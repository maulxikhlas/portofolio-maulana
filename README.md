# Portofolio M Maulana Ikhlas

Ini adalah *source code* untuk situs portofolio pribadi saya. Dibangun dengan fokus pada efisiensi, performa, dan desain minimalis untuk menonjolkan pengalaman serta proyek di bidang **Cybersecurity** dan **Web Development**.

## Tech Stack
* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Language:** TypeScript
* **Deployment:** Vercel

## Fitur
* **Resume-style Layout:** Desain minimalis untuk kemudahan akses informasi bagi *recruiter*.
* **Dark Mode:** Tema gelap yang estetik dan nyaman untuk dibaca.
* **Dynamic Export:** Fitur konversi halaman ke PDF untuk keperluan profesional.
* **Responsive Design:** Optimal di perangkat seluler maupun desktop.
